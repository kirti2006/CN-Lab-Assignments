# Computer Networks Lab Assignments (ENCS352)

This repository contains lab assignments for the Computer Networks course. All experiments are performed using Cisco Packet Tracer.

## 📌 Assignment 1 – Basic Network Topologies

- Implemented Star, Hub-based, and Ring-like topologies
- Tested communication using ICMP ping
- Observed performance and fault tolerance differences

---

## 📌 Assignment 2 – Packet Flow Visualization

- Created a simple LAN network
- Observed ARP and ICMP packet flow using Simulation Mode
- Analyzed switch MAC address learning behavior

---

## 🧪 How to Run

1. Open Cisco Packet Tracer.
2. Open the required `.pkt` file from any assignment folder.
3. Click on a PC → Desktop → Command Prompt.
4. Run ping commands.
5. Switch to Simulation Mode to observe packet flow.
